# 🛢️ PetroVisual Pro: Next-Gen Well Log Interpreter

PetroVisual Pro is a high-performance web application designed for geoscientists and petrophysicists to visualize and interpret borehole data in real-time.

## 🚀 Features
- **Instant Visualization:** Upload `.xlsx` or `.csv` well logs and view plots instantly.
- **Dynamic Lithology Shading:** Interactive sliders to define Sand/Shale cutoffs.
- **Automated Vshale:** Real-time calculation of the Volume of Shale.
- **Export Ready:** Processed data can be downloaded for further geological analysis.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Framework:** Streamlit
- **Data Science:** Pandas, NumPy
- **Visualization:** Matplotlib

## 📖 How to Use
1. Upload your log data (Columns: Depth, Gamma Ray).
2. Adjust the **GR Clean** and **GR Shale** values in the sidebar.
3. Slide the **Cutoff** to see the lithology interpretation change live.
4. Download your results!
